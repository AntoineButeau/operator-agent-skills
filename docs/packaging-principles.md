# Packaging Principles

- One pack maps to one source content series.
- Each pack contains practical operator workflows, not summaries of essays.
- Skills must have clear inputs, steps, outputs, review gates, failure modes, and an example prompt.
- Packs stay portable: Markdown for humans and agents, YAML/JSON for indexes and conversion.
- Human review remains mandatory for external publishing, personnel decisions, legal/security matters, customer commitments, and system-of-record changes.
