# PLG Operating Pack

## Source series
Product-Led Growth

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert product-led growth material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Activation Path Audit](skills/activation-path-audit.md) — diagnose the current state and name the operating risk.
2. [Self-Serve Conversion Map](skills/self-serve-conversion-map.md) — make the system visible by mapping owners, flows, and evidence.
3. [Product-Qualified Signal Design](skills/product-qualified-signal-design.md) — produce a reusable artifact or operating spec.
4. [PLG Experiment Brief](skills/plg-experiment-brief.md) — compress the situation into a decision-ready brief.
