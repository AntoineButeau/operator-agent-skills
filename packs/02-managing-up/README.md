# Upward Signal Pack

## Source series
Managing Up

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert managing up material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Manager Trust Update](skills/manager-trust-update.md) — compress the situation into a decision-ready brief.
2. [Upward Disagreement Brief](skills/upward-disagreement-brief.md) — compress the situation into a decision-ready brief.
3. [Escalation Judgment Check](skills/escalation-judgment-check.md) — diagnose the current state and name the operating risk.
4. [Priority Reading Map](skills/priority-reading-map.md) — make the system visible by mapping owners, flows, and evidence.
