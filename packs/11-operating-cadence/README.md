# Operating Cadence Pack

## Source series
Operating Cadence & Management Systems

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to design operating reviews that convert signals into decisions instead of meeting load. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Cadence Architecture Map](skills/cadence-architecture-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Weekly Business Review Design](skills/weekly-business-review-design.md) — diagnose the current state and name the operating risk.
3. [Meeting System Prune](skills/meeting-system-prune.md) — handle the prune step without overlapping the other skills in the pack.
4. [Cadence Health Check](skills/cadence-health-check.md) — diagnose the current state and name the operating risk.
