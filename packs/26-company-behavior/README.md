# Behavioral OS Pack

## Source series
How Companies Actually Behave: Operating Principles, Decision Rules, and the Hidden System Behind Culture

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert how companies actually behave: operating principles, decision rules, and the hidden system behind culture material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Revealed Behavior Audit](skills/revealed-behavior-audit.md) — diagnose the current state and name the operating risk.
2. [Decision Rule Extractor](skills/decision-rule-extractor.md) — handle the extractor step without overlapping the other skills in the pack.
3. [Culture Operating Gap Review](skills/culture-operating-gap-review.md) — diagnose the current state and name the operating risk.
4. [Behavior Change Lever Map](skills/behavior-change-lever-map.md) — make the system visible by mapping owners, flows, and evidence.
