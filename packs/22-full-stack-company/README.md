# Vertical Integration Pack

## Source series
The Full-Stack Company: Vertical Integration in the AI Era

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert the full-stack company: vertical integration in the ai era material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Integration Boundary Audit](skills/integration-boundary-audit.md) — diagnose the current state and name the operating risk.
2. [Control Point Map](skills/control-point-map.md) — make the system visible by mapping owners, flows, and evidence.
3. [Build Buy Partner Brief](skills/build-buy-partner-brief.md) — compress the situation into a decision-ready brief.
4. [Vertical Integration Roadmap](skills/vertical-integration-roadmap.md) — make the system visible by mapping owners, flows, and evidence.
