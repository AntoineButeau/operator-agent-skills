# Strategy Translation Pack

## Source series
Strategy Execution

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert strategy execution material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Strategy to Work Map](skills/strategy-to-work-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Initiative Priority Check](skills/initiative-priority-check.md) — diagnose the current state and name the operating risk.
3. [Execution Dependency Map](skills/execution-dependency-map.md) — make the system visible by mapping owners, flows, and evidence.
4. [Strategy Review Cadence](skills/strategy-review-cadence.md) — diagnose the current state and name the operating risk.
