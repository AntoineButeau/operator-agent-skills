# AI Context Layer Pack

## Source series
The AI Context Layer: Giving Agents a Business Model They Can Safely Use

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert the ai context layer: giving agents a business model they can safely use material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Agent Context Inventory](skills/agent-context-inventory.md) — handle the system step without overlapping the other skills in the pack.
2. [Context Contract Design](skills/context-contract-design.md) — handle the system step without overlapping the other skills in the pack.
3. [Context Risk Review](skills/context-risk-review.md) — diagnose the current state and name the operating risk.
4. [Business Model for Agents Brief](skills/business-model-for-agents-brief.md) — produce a reusable decision brief or operating spec.
