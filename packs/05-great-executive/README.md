# Executive Attention Pack

## Source series
Being a Great Executive

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert being a great executive material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Executive Attention Audit](skills/executive-attention-audit.md) — diagnose the current state and name the operating risk.
2. [Strategic No Brief](skills/strategic-no-brief.md) — compress the situation into a decision-ready brief.
3. [Talent Calibration Loop](skills/talent-calibration-loop.md) — handle the loop step without overlapping the other skills in the pack.
4. [Legacy System Review](skills/legacy-system-review.md) — diagnose the current state and name the operating risk.
