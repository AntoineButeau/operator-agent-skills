# Taste Bottleneck Pack

## Source series
Taste Is the Bottleneck

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert taste is the bottleneck material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Taste Gap Diagnosis](skills/taste-gap-diagnosis.md) — handle the system step without overlapping the other skills in the pack.
2. [Reference Set Builder](skills/reference-set-builder.md) — handle the system step without overlapping the other skills in the pack.
3. [Quality Bar Review](skills/quality-bar-review.md) — diagnose the current state and name the operating risk.
4. [Taste Calibration Loop](skills/taste-calibration-loop.md) — handle the system step without overlapping the other skills in the pack.
