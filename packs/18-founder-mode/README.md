# Founder Operating Mode Pack

## Source series
Founder Mode That Actually Scales

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert founder mode that actually scales material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Founder Bottleneck Audit](skills/founder-leverage-audit.md) — diagnose the current state and name the operating risk.
2. [Founder Context Capture](skills/founder-context-capture.md) — handle the capture step without overlapping the other skills in the pack.
3. [Founder Decision Rights Reset](skills/founder-decision-rights-reset.md) — handle the reset step without overlapping the other skills in the pack.
4. [Scaling Founder Rituals](skills/scaling-founder-rituals.md) — handle the rituals step without overlapping the other skills in the pack.
