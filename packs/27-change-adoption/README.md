# Change Adoption Pack

## Source series
Change That Actually Takes: How Companies Move From Announcement to Adoption

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert change that actually takes: how companies move from announcement to adoption material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Change Readiness Scan](skills/change-readiness-scan.md) — diagnose the current state and name the operating risk.
2. [Announcement to Adoption Plan](skills/announcement-to-adoption-plan.md) — turn the decision into sequenced work with gates and owners.
3. [Resistance Diagnosis](skills/resistance-diagnosis.md) — diagnose the current state and name the operating risk.
4. [Change Embedding Review](skills/change-embedding-review.md) — diagnose the current state and name the operating risk.
