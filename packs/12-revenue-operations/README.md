# RevOps System Pack

## Source series
Revenue Operations

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to make revenue motion inspectable from handoff to forecast commit. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Funnel Integrity Audit](skills/funnel-integrity-audit.md) — diagnose the current state and name the operating risk.
2. [GTM Handoff Map](skills/gtm-handoff-map.md) — make the system visible by mapping owners, flows, and evidence.
3. [Forecast Reliability Review](skills/forecast-reliability-review.md) — diagnose the current state and name the operating risk.
4. [RevOps Change Control](skills/revops-change-control.md) — handle the control step without overlapping the other skills in the pack.
