# Network Effects Pack

## Source series
Network Effects That Actually Compound

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert network effects that actually compound material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Network Loop Map](skills/network-loop-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Cold Start Risk Scan](skills/cold-start-risk-scan.md) — surface risks, constraints, and missing evidence before action.
3. [Liquidity Metric Design](skills/liquidity-metric-design.md) — design the operating artifact, interface, or workflow contract.
4. [Defensibility Review](skills/defensibility-review.md) — diagnose the current state and name the operating risk.
