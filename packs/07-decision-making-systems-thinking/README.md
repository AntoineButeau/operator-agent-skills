# Judgment System Pack

## Source series
Decision Making & Systems Thinking

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert decision making & systems thinking material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Decision Type Classifier](skills/decision-type-classifier.md) — handle the classifier step without overlapping the other skills in the pack.
2. [Second-Order Effects Scan](skills/second-order-effects-scan.md) — diagnose the current state and name the operating risk.
3. [Constraint Finder](skills/constraint-finder.md) — handle the finder step without overlapping the other skills in the pack.
4. [Incentive Behavior Review](skills/incentive-behavior-review.md) — diagnose the current state and name the operating risk.
