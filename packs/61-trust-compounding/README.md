# Trust Compounding Pack

## Source series
Trust That Actually Compounds

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert trust that actually compounds material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Trust Signal Map](skills/trust-signal-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Commitment Reliability Review](skills/commitment-reliability-review.md) — diagnose the current state and name the operating risk.
3. [Trust Repair Plan](skills/trust-repair-plan.md) — turn the diagnosis into an executable plan with owners and gates.
4. [Compounding Trust Loop](skills/compounding-trust-loop.md) — handle the system step without overlapping the other skills in the pack.
