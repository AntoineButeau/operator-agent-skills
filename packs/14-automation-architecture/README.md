# Automation Design Pack

## Source series
Automation / AI Automation Architecture

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert automation / ai automation architecture material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Automation Opportunity Scan](skills/automation-opportunity-scan.md) — diagnose the current state and name the operating risk.
2. [Human-in-the-Loop Design](skills/human-in-the-loop-design.md) — produce a reusable artifact or operating spec.
3. [Automation ROI Brief](skills/automation-roi-brief.md) — compress the situation into a decision-ready brief.
4. [Agent Workflow Spec](skills/agent-workflow-spec.md) — produce a reusable artifact or operating spec.
