# Competitive Strategy Pack

## Source series
Competitive Strategy

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert competitive strategy material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Competitive Position Map](skills/competitive-position-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Moat Evidence Review](skills/moat-evidence-review.md) — diagnose the current state and name the operating risk.
3. [Competitive Response Brief](skills/competitive-response-brief.md) — produce a reusable decision brief or operating spec.
4. [Strategy War Game](skills/strategy-war-game.md) — handle the system step without overlapping the other skills in the pack.
