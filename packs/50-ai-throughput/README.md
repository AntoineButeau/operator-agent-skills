# AI Throughput Pack

## Source series
From Productivity to Throughput

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert from productivity to throughput material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Throughput Constraint Map](skills/throughput-constraint-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Queue Health Review](skills/queue-health-review.md) — diagnose the current state and name the operating risk.
3. [Agent Leverage Plan](skills/agent-leverage-plan.md) — turn the diagnosis into an executable plan with owners and gates.
4. [Throughput Metric Design](skills/throughput-metric-design.md) — design the operating artifact, interface, or workflow contract.
