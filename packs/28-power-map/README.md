# Power Map Pack

## Source series
Power That Actually Moves Work

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert power that actually moves work material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Influence Network Map](skills/influence-network-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Power Risk Review](skills/power-risk-review.md) — diagnose the current state and name the operating risk.
3. [Coalition Building Plan](skills/coalition-building-plan.md) — turn the decision into sequenced work with gates and owners.
4. [Political Friction Diagnosis](skills/political-friction-diagnosis.md) — diagnose the current state and name the operating risk.
