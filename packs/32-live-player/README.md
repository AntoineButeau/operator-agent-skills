# Reality Contact Pack

## Source series
How to Be a Live Player

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert how to be a live player material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Reality Contact Audit](skills/reality-contact-audit.md) — diagnose the current state and name the operating risk.
2. [Assumption Refresh Loop](skills/assumption-refresh-loop.md) — handle the loop step without overlapping the other skills in the pack.
3. [Live Option Generator](skills/live-option-generator.md) — handle the generator step without overlapping the other skills in the pack.
4. [Field Signal Review](skills/field-signal-review.md) — diagnose the current state and name the operating risk.
