# Operating Review Pack

## Source series
Operating Reviews That Actually Drive Decisions

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert operating reviews that actually drive decisions material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Operating Review Agenda Builder](skills/operating-review-agenda-builder.md) — diagnose the current state and name the operating risk.
2. [Decision Followthrough Audit](skills/decision-followthrough-audit.md) — find gaps between stated process and observed behavior.
3. [Metric Narrative Review](skills/metric-narrative-review.md) — diagnose the current state and name the operating risk.
4. [Review Cadence Redesign](skills/review-cadence-redesign.md) — design the operating artifact, interface, or workflow contract.
