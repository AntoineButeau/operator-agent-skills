# Right Depth Pack

## Source series
The Right Depth for the Problem

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert the right depth for the problem material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Depth Selection Check](skills/depth-selection-check.md) — handle the system step without overlapping the other skills in the pack.
2. [Analysis Scope Cutter](skills/analysis-scope-cutter.md) — handle the system step without overlapping the other skills in the pack.
3. [Precision Risk Review](skills/precision-risk-review.md) — diagnose the current state and name the operating risk.
4. [Decision Depth Brief](skills/decision-depth-brief.md) — produce a reusable decision brief or operating spec.
