# AI Control Plane Pack

## Source series
The AI Control Plane

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert the ai control plane material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Agent Registry Map](skills/agent-registry-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Permission Boundary Review](skills/permission-boundary-review.md) — diagnose the current state and name the operating risk.
3. [Control Plane Operating Policy](skills/control-plane-operating-policy.md) — handle the system step without overlapping the other skills in the pack.
4. [Agent Incident Retrospective](skills/agent-incident-retrospective.md) — extract lessons and update the operating system.
