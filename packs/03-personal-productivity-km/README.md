# Personal Knowledge OS Pack

## Source series
Personal Productivity + Knowledge Management

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert personal productivity + knowledge management material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Capture Purpose Filter](skills/capture-purpose-filter.md) — handle the filter step without overlapping the other skills in the pack.
2. [Weekly Review Protocol](skills/weekly-review-protocol.md) — diagnose the current state and name the operating risk.
3. [Notes to Output Pipeline](skills/notes-to-output-pipeline.md) — handle the pipeline step without overlapping the other skills in the pack.
4. [System Decay Triage](skills/system-decay-triage.md) — handle the triage step without overlapping the other skills in the pack.
