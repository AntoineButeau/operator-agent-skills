# Manager Operating System Pack

## Source series
Expert Manager

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert expert manager material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [First 90 Days Manager Diagnostic](skills/first-90-days-manager-diagnostic.md) — handle the diagnostic step without overlapping the other skills in the pack.
2. [Delegation Contract Builder](skills/delegation-contract-builder.md) — produce a reusable artifact or operating spec.
3. [One-on-One System Designer](skills/one-on-one-system-designer.md) — produce a reusable artifact or operating spec.
4. [Decision Journal Coach](skills/decision-journal-coach.md) — handle the coach step without overlapping the other skills in the pack.
