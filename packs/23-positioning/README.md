# Positioning System Pack

## Source series
Positioning

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert positioning material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Category Context Map](skills/category-context-map.md) — make the system visible by mapping owners, flows, and evidence.
2. [Positioning Statement Builder](skills/positioning-statement-builder.md) — produce a reusable artifact or operating spec.
3. [Proof Point Inventory](skills/proof-point-inventory.md) — make the system visible by mapping owners, flows, and evidence.
4. [Message Confusion Audit](skills/message-confusion-audit.md) — diagnose the current state and name the operating risk.
