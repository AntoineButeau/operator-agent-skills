# Agent-Native Tools Pack

## Source series
Agent-Native Tools

## Who it is for
Operators, founders, executives, managers, chiefs of staff, BizOps leads, product leaders, revenue leaders, and AI agents helping them make decisions from live operating context.

## When to use this pack
Use this pack to convert agent-native tools material into operator-grade decisions and reusable artifacts. The four skills are deliberately sequenced so diagnosis, mapping, planning, and review do not collapse into one vague prompt.

## Progression / use order
1. [Agent Tool Fit Check](skills/agent-tool-fit-check.md) — handle the system step without overlapping the other skills in the pack.
2. [Tool Context Contract](skills/tool-context-contract.md) — handle the system step without overlapping the other skills in the pack.
3. [Agent UX Review](skills/agent-ux-review.md) — diagnose the current state and name the operating risk.
4. [Tool Evaluation Plan](skills/tool-evaluation-plan.md) — turn the diagnosis into an executable plan with owners and gates.
